```java
## Co to je tedy if ?

- do If statemantu zadáváme podmínky které chceme aby se splnily
- například

int a = 0;
if(a==1){ // podmínka která má být splněna 
sout ("a se rovná jedna"); // úkon který má udělat když je podmínka pravnivá
}else { // jinak
sout ("a se nerovna 1") //úkon který má udělat když je podmínka pravnivá
}
// nebo také můžeme udělat if statement u else
// následovně
if(a==1){  
sout ("a se rovná jedna"); 
}else if(a==0){ //další if statement zavádí podmínku pokud by nevyšla předchozí podmínka
sout ("a se rovna nula)
}else{ //na konec opět else
sout ("a se nerovná nula") 
}
```
